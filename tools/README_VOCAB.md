## Usage

These vocabulary files are input to `generate_espeak_dict.py`, which uses espeak-ng
to generate IPA pronunciations. The resulting dictionaries contain:

- **Input**: Word lists from FrequencyWords (MIT)
- **Output**: IPA pronunciations generated by espeak-ng (GPL, used as tool only)
- **Embedded**: IPA pronunciation data (facts, not copyrightable code)

The generated dictionaries are pure data derived from espeak-ng's pronunciation
engine and can be embedded under EthervoxAI's license as they contain no GPL code.

## Regenerating Dictionaries

```bash
# Download fresh vocabulary (top 10,000 words)
curl -sL "https://raw.githubusercontent.com/hermitdave/FrequencyWords/master/content/2018/de/de_50k.txt" \
  | head -10000 | awk '{print $1}' > tools/vocab_de_large.txt

curl -sL "https://raw.githubusercontent.com/hermitdave/FrequencyWords/master/content/2018/es/es_50k.txt" \
  | head -10000 | awk '{print $1}' > tools/vocab_es_large.txt

# Generate dictionaries with C headers
python3 tools/generate_espeak_dict.py \
  --lang de \
  --vocab-file tools/vocab_de_large.txt \
  --output src/tts/phonemizer/data/espeak_dict_de.dict \
  --format dict

python3 tools/generate_espeak_dict.py \
  --lang de \
  --dict-file src/tts/phonemizer/data/espeak_dict_de.dict \
  --output src/tts/phonemizer/data/espeak_dict_de.h \
  --format header

# Repeat for Spanish
python3 tools/generate_espeak_dict.py \
  --lang es-419 \
  --vocab-file tools/vocab_es_large.txt \
  --output src/tts/phonemizer/data/espeak_dict_es_419.dict \
  --format dict

python3 tools/generate_espeak_dict.py \
  --lang es-419 \
  --dict-file src/tts/phonemizer/data/espeak_dict_es_419.dict \
  --output src/tts/phonemizer/data/espeak_dict_es_419.h \
  --format header
```
